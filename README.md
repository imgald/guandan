# 掼蛋

当前仓库包含两部分：

- 本地模式：1 名玩家对 3 个 AI
- 联网模式：创建房间、加入房间、房主开始、房间聊天、2 人自动补 AI 搭档、3-4 人随机座位与组队

## 运行

推荐直接启动 Node 服务：

```powershell
cd E:\guandan
node server.js
```

或者：

```powershell
cd E:\guandan
npm start
```

然后打开：

- [http://localhost:8000](http://localhost:8000)
